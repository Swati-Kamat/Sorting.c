# Sorting.c
  What is Sorting?
Sorting is the process of arranging data in a specific order, usually ascending (small → large) or descending (large → small).
Sorting is used to make data easier to search, analyze, and manage.
  Types of Sorting
Sorting algorithms are different methods used to arrange data.
  1. Bubble Sort
Compares adjacent elements and swaps them if they are in the wrong order.
Repeats until the list is sorted.
Simple but slow for large data.
  2. Selection Sort
Finds the smallest element in the list.
Places it at the beginning.
Repeats for the remaining elements.
  3. Insertion Sort
Takes one element and inserts it into the correct position in the sorted part of the list.
Works similar to sorting playing cards in hand.
  4. Merge Sort
Uses divide and conquer.
Splits the array into smaller parts, sorts them, and merges them together.
Efficient for large data
  5. Quick Sort
Picks a pivot element.
Places smaller elements before it and larger elements after it.
Very fast and widely used.
